## Min & Max Dates

#### Overview
Restrict the range of selectable dates with the minDate and maxDate options. Set the beginning and end dates

#### HTML Structure
```html
    <div class="hello-week"></div>
```

#### Javascript Initialization
```js
    new HelloWeek({
        minDate: "2019-03-10",
        maxDate: "2019-12-26"
    });
```
